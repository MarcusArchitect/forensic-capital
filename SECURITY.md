# Security Policy

## Scope

This repository contains security research tooling and analysis
methodology published by Forensic Capital for educational and
defensive purposes. The tools inspect publicly observable
configuration and parameter data only. They do not exploit,
attack, or interact with live systems without authorization.

## Responsible Disclosure

Forensic Capital follows coordinated disclosure. Findings against
third-party protocols are reported privately through authorized
programs (Immunefi, HackerOne, Bugcrowd) before any public
discussion.

## Reporting a Vulnerability in This Repository

Do not open a public issue.
Email research@forensic-capital.com with:
- affected file and line
- description and impact
- reproduction steps

We acknowledge within 72 hours.

## Disclosure Timeline

- T+0     : Report received, acknowledged within 72h
- T+0-90d : Coordinated remediation window
- T+90d   : Public disclosure by default,
            or earlier with consent,
            or extended by mutual agreement

## Out of Scope

Using these tools against systems you are not authorized to
test is solely your responsibility.
Demo output (Mode: [DEMO]) is not a live finding.
